# 디자인 언어: 모던 SaaS / Linear 풍

aesthetic-system 보조 레퍼런스. director가 이 언어를 골랐을 때 로드한다.

## 정신

Linear, Vercel, Stripe, Arc로 대표되는 동시대 프로덕트 미감. **세련된 깊이, 섬세한 디테일, 절제된 화려함.** 미니멀의 규율 위에 미묘한 그라데이션·글로우·유리질감으로 '고급스러운 깊이'를 더한다.

## 색

- **무채색 베이스 + 절제된 브랜드 색** + 미묘한 그라데이션. 미니멀보다 색을 약간 더 쓰되 여전히 차분.
- **서피스 레이어링**: surface.base / raised / sunken을 명도 2~4% 차로 분리해 카드·패널의 깊이 표현.
- **미묘한 글로우·그라데이션**: 버튼·히어로에 같은 색의 약한 그라데이션, accent 주변 soft glow. 과하지 않게.
- 다크모드가 1급 시민. 순수 검정 대신 살짝 든 회색(#0a0a0c 류) + 채도 낮춘 색.

## 타이포

- Inter, Geist, SF 류의 현대적 그로테스크. x-height 크고 가독성 높은 본문.
- 큰 디스플레이 타이포 + 촘촘한 자간(헤드라인 -0.02em 류)으로 현대적 인상.
- 본문은 차분, 헤드라인에서 대비.

## 레이아웃·디테일

- border-radius 중간(6~12px). 카드·버튼·인풋에 일관 적용.
- **그림자는 다층·미묘**: 0 1px 2px + 0 4px 12px 류의 여러 겹 soft shadow로 자연광 느낌.
- 1px hairline border(반투명 흰/검)로 요소 경계.
- 글래스모피즘은 절제해서: 오버레이·내비에 backdrop-blur + 반투명.

## 모션

- 부드럽고 빠른 마이크로 인터랙션(150~250ms, ease-out). hover 시 미묘한 lift/glow.
- spring 류의 자연스러운 이징. 과한 바운스는 피함.

## 피해야 할 것

- 과한 채도·무지개색, 두꺼운 하드 보더, 평면적 단색 그림자, 느리고 과장된 모션.
